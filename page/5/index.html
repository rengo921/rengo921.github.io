5個最佳線上檔案共享網站

lifehacker.com 最近從200個提名中評選出最好的5個線上檔案共享服務，這些容量大、速度快、容易使用的線上服務可以幫助你和朋友一起分享音樂，也可以幫助你和同事共享某個重要文件。

Box.net

提供1G免費空間的 Box.net 以其簡潔的介面以及豐富的功能著稱。使用者喜歡它的共同工具與開放格式，並因此享受直接在桌面安裝 Web 空間的樂趣，該網站支援Web服務以及RSS feeds。Box.net 提供直接檔案連結網址，你不必登入就可以直接使用這個網址下載，Box.net 最大缺點是，免費使用者的單一檔案容量上限是10M。

Drop.io

Drop.io 簡潔的頁面讓人印象深刻。不必註冊，內建不同格式文件的預覽工具，該網站最近新增免費傳真收發，錄音等功能，免費帳號的容量上限是100M。

MediaFire

人們喜歡無限空間，MediaFire 就是。服務是免費的，空間無限制，不必註冊，但單個檔不能超過100M。

Windows Live SkyDrive

Windows Live SkyDrive 提供5GB免費空間，這個來自微軟的服務一經上線立刻備受歡迎。SkyDrive 提供資料夾的共享服務，你可以將檔放在個人、共享、公開資料夾中。SkyDrive 使用 Windows Live 帳號登入，單一檔案上限是50M。

FolderShare

跨平台的 FolderShare 更像一個檔案同步工具，你可以直接在桌面上和朋友共享文件，在桌面建立一個共享資料夾，該服務可以直接將這個資料夾中的檔案同步到你的朋友的桌面，你所做的只是將需要共享的檔案拖放到這個共享資料夾。

Google site開放使用！

其實也不是太新的玩意了，原本是在Google Apps底下的服務，現在也開放給一般使用者使用 (當然首先你得先申請一組Google的帳號密碼)。

那麼Google site有什麼樣的特色呢？官方網站註記了以下幾點說明︰

‧滑鼠單點即可建立頁面
‧不需具備HTML語法能力 (當然會一點更棒)
‧個人化視覺設定
‧資料共享
‧是免費的！

連線到Google site網站並且登入Google帳號，接下來選擇「Creat new site」建立一個新網站。

輸入網域名稱 (例如︰XXOO，那麼實際網址就會變成http://sites.google.com/site/sites/**XXOO**)，接著填上網頁內容簡述，設定是否開放給所有人觀看，或者是只有指定的使用者才能進入。最後再選擇網頁外觀風格之後，按下「Creat site」就完成網站初步建立。

接下來，進入編輯頁面繼續建立網站各單篇網頁內容，這邊的操作模式有點類似撰寫部落格的方式，可以簡單地打打字、按按滑鼠就完成頁面，當然也支援自行輸入Html語法。

方便強大的 Firefox Universal Uploader

上下傳 Flickr、Picasa、YouTube 的檔案 - 方便強大的 Firefox Universal Uploader

今天挖到「Firefox Universal Uploader (以下簡稱「FireUploader」)」這個好物實在讓我太感動了 Q_Q, 所以迫不及待的就算犧牲午睡也要趕快的介紹給大家。

「FireUploader」是個讓你可以很方便的上、下傳檔案到一些知名 Web 2.0 網站的 Firefox 擴充套件。目前支援「Flickr」、「Picasa」、「YouTube」、「Box.net」:

那麼,「FireUploader」究竟有什麼好用的地方呢!? 以下就以「Flickr」為例做說明。

當你設定好「Flickr」帳號並登入後, 可以發現「FireUploader」的視窗分為了三大塊: 左邊的是用來顯示本地端檔案及目錄、右邊則是遠端的檔案、下方則是用來顯示傳輸進度及資訊看板。關於資訊看板以「Flickr」來說資訊看板就是讓你編輯相片的標題, 描述及設定權限之用。

注意到了嗎? 右方遠端檔案的地方居然有「目錄」的存在, 這個「目錄」就是你在「Flickr」所建立的「SET」, 點進去「目錄」後就可以看到在該「SET」的相片。「FireUploader」這種類似 Windows 檔案總管及 FTP 概念的設計, 讓上傳及備份相片變得很容易。

舉例來說: 如果我們想備份某一本「SET」的相片, 我們只要對著相對應的目錄按右鍵, 然後選擇下載即可; 下載回來的檔案就會自動放在以該「SET」名稱命名的目錄下, 而檔案名稱則會是該相片檔案在「Flickr」上「Title」欄位的描述 (「Title」重覆的會自動加上流水號), 夠方便吧!

那麼上傳呢? 如果我們想上傳一堆相片, 我們也可以把相片都放在一個目錄中, 然後點選目錄再進行上傳, 如此一次就能上傳該目錄中所有的相片。只是, 有點可惜的是當你上傳整個目錄時,「FireUploader」並不會自動幫你建立「SET」, 希望之後的版本能加上這個功能。
