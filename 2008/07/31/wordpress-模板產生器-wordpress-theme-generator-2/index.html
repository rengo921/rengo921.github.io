WordPress 模板產生器 Wordpress Theme Generator

今天又逛到一個好東西，網路上有人提供一個『WordPress 模板產生器 Wordpress Theme Generator』，它可以自己產生LOGO、自動產生EMail圖片、不需要任何HTML, JS, PHP, CSS的網頁知識，以WordPress 2.1為基礎。

只要將你想顯示的條件設定即可，按下『Preview』就可先預覽，其實也不用按『Preview』，這個網站有即時預覽的功能，設定完畢後按下『Save』，它會顯示這個模版所有的檔案，你可以分開下載，或者網站也提供ZIP壓縮檔下載，真的很不錯！

有這種東西就不需要再為了去尋找漂亮或符合自己的模板傷腦漿了！缺點就是你多少要懂點英文！
