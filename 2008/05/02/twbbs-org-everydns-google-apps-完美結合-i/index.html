twbbs.org / everydns / google apps 完美結合 I

twbbs.org 改版後增加了一些重要的功能，讓我能成功整合 Google APPs 又省去繁瑣的 DNS 設定。關於 Google Apps 的設定網上有一堆，找一下就有了。而純敏達康中有說明一些設定的概念，可以參考一下。
就讓我來說明如何設定：
1. 註冊新 twbbs 網域， 你需要一個申請一個帳號，用來註冊網域。有三個網域可選擇：
twbbs.org, twgg.org, cnkk.org
我註冊的就是 yp312.twbbs.org
2. 使用免費的 DNS 主機服務，管理您的 twbbs.org
將 ns1.everydns.net 及 ns2.everydns.net 的 ip 填入，並在你的 domain 前加上 ns1 / ns2
設定好按更新

3. 設定 everydns.net
http://www.everydns.net/ 申請帳號。
在左側的 add new domain 內輸入你的 xxx.twbbs.org 的 domain.
好了按 >> basic

在新增 Domain 區上面那個區塊的 primary domains 會出現你剛輸入的 xxx.twbbs.org.
點一下 xxx.twbbs.org 就可以在右邊的區塊更改詳細的設定（如下下圖）。

一開始會出現兩列設定，在 xxx.twbbs.org 的 value 內會顯示你現在的 ip, 而 www.xxx.twbbs.org 系統會自訂轉到 parked.everydns.net ，這兩列之後都要刪掉，是沒有用的。

你可以在 add a record 區內輸入你需要的資料後，點 add record 後，你輸的東西就會出現在上面的區塊中。注意一點，xxx.twbbs.org 的 record type 一定要是 A, 而 record value 是他的 ip. 若是 dyanmic ip 需要查一下如何設才行（我有個固定 ip 可用）。
或許可以設成 66.249.81.121 (ghs.google.com 的 ip…，我沒試…不敢保證，有試的人可告訴我結果。）

4. 啟用並設定 Google APPs
啟用：http://www.google.com/a/help/intl/zh-TW/admins/editions_spe.html
Google Apps 申請教學
因為 DNS 的更新還算快，所以你可以選 cname 的方式更新。
domain verify 後，就算完成 90%. 剩下就看你要不要把 google 的各個 service 都弄個 domain name 如 mail.xxx.twbbs.org 之類的。你可以參考我下面的設定
5. 下面是我所有的設定。可以先把 dns 設好，再去 active 各個 google service 並 modify url 改成你要的 domain.

6. mail 的設定可參考如下：就是將 record type 改成 MX，而你的 domain 就是 xxx.twbbs.org
設完過幾分之後就可以寄信看看。
